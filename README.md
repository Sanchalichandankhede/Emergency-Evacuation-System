BFS vs DFS – Evacuation System

Overview

This project uses **BFS** and **DFS** to simulate an emergency evacuation system using a graph (rooms = nodes, corridors = edges).

 Purpose

BFS → Find shortest path to exit
DFS → Explore full building



 Run

```bash
gcc bfs.c -o bfs
gcc dfs.c -o dfs

./bfs
./dfs
```



 Complexity

* BFS: O(V + E), Space: O(V)
* DFS: O(V + E), Space: O(V)
